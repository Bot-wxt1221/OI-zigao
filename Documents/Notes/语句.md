# 语句/算法

## rand

库:`<cstdio>`

函数:`rand()`

返回:`int`



### 用法:

#### 随机种子

```c++
srand((unsigned int)time(NULL));
```

e.g.使用时间戳

```c++
#include<cstdlib>
#include<ctime>

using namespace std;
int main(){
	srand((unsigned int)time(NULL));//随机种子
    
	return 0;
}
```

#### 调用

```c++
int n=rand();
```

#### 特定范围

(a, b)

```c++
int n=rand()%(b-a-1)+a+1
```

(a, b]

```c++
int n=rand()%(b-a)+a+1
```

[a, b)

```c++
int n=rand()%(b-a)+a
```

[a, b]

```c++
int n=rand%(b-a+1)+a
```

## 数组

### 用法

#### 申请内存

```c++
<type> <name>[<leigh>]
```

#### 初始化

```c++
//方法1
<type> <name>[leigh]={_type_,_type_,_type_...}

//方法1
<type> <name>[leigh]
<name>[下标]={_type_}
```

### 注意

a[0]~a[n]

```c++
for(int i=0;i<SIZE;i++){
	scanf("%c",&a[i]);//所有字符被输入包括空格 换行
	cin >> a[i];//空格 换行分割
}
```

二维素组

```c++
a[i][j];
```

内存等价于

```
a[i*j];
```
